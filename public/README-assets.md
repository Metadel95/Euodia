# Public Assets

## Required Files

Replace these placeholders before deploying:

### Video
- `hero-video.mp4` — Full-screen background video for the hero section.
  - Recommended: 1920×1080 or higher, H.264, under 15MB
  - Should evoke worship, nature, candlelight, or community gatherings
  - Must be muted-safe (no important audio)

### Images
- `hero-poster.jpg` — Static fallback shown before/if the video loads (1920×1080)
- `og-image.jpg` — Open Graph share image (1200×630)

### Favicon / Icons
- `favicon.ico` — Browser tab icon
- `favicon-16x16.png` — 16×16 favicon
- `favicon-32x32.png` — 32×32 favicon  
- `apple-touch-icon.png` — 180×180 Apple touch icon
- `android-chrome-192x192.png` — 192×192 Android icon
- `android-chrome-512x512.png` — 512×512 Android icon

## Tool Recommendations

- Favicon generation: https://realfavicongenerator.net
- Video compression: HandBrake (free, cross-platform)
- OG image: Figma or Canva with 1200×630 canvas
